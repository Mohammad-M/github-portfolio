A comprehensive, real-time web and mobile application designed to facilitate seamless communication between users and backend services through a proxy architecture. The system is built with scalability, modularity, and performance in mind.

    Frontend:

  • Built with ReactJS, TypeScript, TailwindCSS, HTML/CSS
  • Implements dynamic UI components and responsive design
  • Integrates with backend via RESTful APIs and WebSockets
  • Follows MVC architecture for maintainability and scalability

    • Proxy Server:

  • Acts as a middleware layer between frontend and backend services
  • Handles routing, authentication, and data transformation

  • Ensures secure and efficient data flow across services

  • Enables proactive debugging and system optimization

  • Stores user preferences locally via Shared Preferences
